BMI Calculator Project

### About
This python program calculates BMI for 3 people and saves reports to a text file.

### How to run
1. Open 'bmi.calculator.ipynb' in Jupyter Notebook
2. Run all cells from top to bottom
3. Enter name, age, weight, height for three people when prompted
4. Check 'bmi_reports.txt' for saved results
